# Code of Conduct

## Our Pledge

Orange Music is a graduation project created for learning, research, and demonstration purposes.

We are committed to maintaining a respectful, friendly, and professional environment for everyone who views, uses, or contributes to this project.

## Expected Behavior

Examples of expected behavior include:

- Using respectful and constructive language
- Giving clear and helpful feedback
- Respecting different opinions and experiences
- Reporting issues politely and with enough context
- Avoiding personal attacks, harassment, or offensive comments

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, insults, or discriminatory language
- Personal attacks or public shaming
- Sharing private information without permission
- Spamming issues, pull requests, or discussions
- Using this project to distribute illegal or unauthorized content

## Project Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Discussions
- Documentation
- Any communication related to this project

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block users who violate this Code of Conduct.

## Contact

For questions or concerns, please contact the project owner through GitHub.
